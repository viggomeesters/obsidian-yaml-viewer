# Changelog

All notable changes to YAML Viewer are documented here.

## 0.1.2 - 2026-06-06

- Publish release assets through GitHub Actions with artifact attestations.
- Document clipboard behavior for review transparency.

## 0.1.1 - 2026-06-06

- Show leading document comments before the root YAML node in the tree view.
- Keep raw source highlighting unchanged.

## 0.1.0 - 2026-06-06

- Initial public release.
- Add read-only `.yaml` and `.yml` file view.
- Add tree rendering for maps, sequences, scalars, aliases, comments, and parse errors.
- Add filter/search, outline navigation, copy raw YAML, copy path, expand/collapse, and raw source highlighting.
